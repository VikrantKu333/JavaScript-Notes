Form Validation

Note : -

Name - Minimum 2 & Maximum 20 Alphabet Character

User Name - First Character Must be Include Alphabet and Minimum 2 & Maximum 15 Alphabet, Number and Underscore Character

Email ID - Valid Email ID Format

Mobile Number - Only 10 Digit Number

Password - Minimum 8 Character. The string must contain at least 1 lowercase alphabetical & 1 uppercase alphabetical, 1 numeric and 1 special character "!@#$%^&*?"

Red Colour Border - Is Not Valid

Green Colour Border - Is Valid
